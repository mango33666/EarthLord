# EarthLord 游戏设计文档 (GDD)

## 文档信息
- **游戏名称**: EarthLord（地球领主）
- **游戏类型**: LBS生存建造游戏
- **核心玩法**: 探索圈地 + 资源采集 + 建造养成 + 社交互动
- **美术风格**: 希望朋克（Hopepunk）
- **目标平台**: iOS
- **文档版本**: v1.0
- **创建日期**: 2025-12-27

---

## 1. 游戏概念

### 1.1 高概念（High Concept）
**"用双脚在末日废土上开拓新世界的生存建造游戏"**

在现实世界中行走探索，圈占土地建立领地，搜刮废墟获取资源，建造家园重建文明，与其他幸存者交易合作，最终成为新世界的领主。

### 1.2 游戏支柱（Core Pillars）
1. **真实世界探索** - Walk to Earn机制，走路即游戏
2. **领地开拓** - GPS圈地，现实中的每一步都是你的领土
3. **资源循环** - 探索→采集→建造→消耗→再探索
4. **社交互动** - 基于位置的玩家交流和交易
5. **长期养成** - 建筑升级、成就收集、排行榜竞争

### 1.3 目标玩家画像
- **探索者**: 喜欢户外活动，享受发现新地点的乐趣
- **收集者**: 热衷于收集资源、建筑、成就
- **建造者**: 喜欢规划和建设自己的基地
- **社交玩家**: 享受与其他玩家互动、交易、组队
- **竞争者**: 追求排行榜排名和稀有成就

## 2. 游戏世界观

### 2.1 故事背景

#### 时间线
**2048年** - "终末战争"爆发
- 全球文明在72小时内崩塌
- 99%的人口消失
- 核冬天降临，通讯中断

**2049-2050年** - 黑暗时代
- 幸存者躲藏在地下避难所
- 资源极度匮乏
- 旧世界秩序完全瓦解

**2051年** - 重建纪元开始
- 核冬天结束，阳光重现
- 《新开拓者协议》颁布
- 幸存者开始走出避难所
- 新的文明秩序建立

#### 核心法则：《新开拓者协议》
> **"凡以双足丈量之无主土地，皆为开拓者所有。"**

这是新世界唯一的法则。没有政府，没有军队，只有GPS卫星还在轨道上运行，记录着每个人走过的每一步。

### 2.2 视觉风格 - 希望朋克（Hopepunk）

**核心理念**: 在绝望中寻找希望，在废墟上重建新生

**色彩设计**:
- **主色调**: 废土灰、锈蚀橙、尘埃棕
- **点缀色**: 新生绿、希望蓝、清晨金
- **对比**: 破败与新生的强烈视觉对比

**环境氛围**:
- 破败的建筑上长出绿色植物
- 废弃的街道上闪烁着新的标记
- 阳光穿透云层，照亮前行的路
- UI设计带有末世科技感，但不失温暖

### 2.3 游戏世界规则

1. **领地法则**: 用GPS轨迹圈出的闭合区域即为领地
2. **资源法则**: 真实世界的POI对应不同的资源类型
3. **生存法则**: 必须持续探索获取食物和水，否则角色会死亡
4. **协作法则**: 玩家可以自由交易，鼓励互助共赢

## 3. 核心游戏循环

### 3.1 短期循环（15-30分钟）
```
出门探索 → 发现POI → 搜刮资源 → 查看背包
    ↓
使用资源恢复体征 → 继续探索或回家建造
```

### 3.2 中期循环（1-3天）
```
圈占新领地 → 搜集足够资源 → 建造基础设施 →
    ↓
领地扩大 → 解锁新建筑 → 提升生产效率 →
    ↓
与其他玩家交易 → 获取稀有资源 → 建造高级建筑
```

### 3.3 长期循环（1周-1个月）
```
完成成就 → 攀升排行榜 → 建立玩家联盟 →
    ↓
扩张领地版图 → 成为区域领主 → 影响服务器格局
```

## 4. 核心玩法系统

### 4.1 探索系统（Exploration）

#### 4.1.1 GPS圈地玩法
**核心机制**: 像小时候在沙滩上画圈一样，用双脚在现实世界中圈出领地

**操作流程**:
1. 打开App，点击"开始开拓"
2. 开始步行，地图上绘制蓝色轨迹线
3. 绕一圈回到起点附近（< 10米）
4. 系统自动检测闭合，轨迹变为绿色多边形
5. 验证通过后，领地永久属于玩家

**设计理念**:
- **简单直观**: 像画画一样简单，无需复杂操作
- **真实反馈**: GPS实时追踪，每一步都有轨迹
- **成就感**: 看着自己圈出的领地，满满的成就感
- **策略性**: 选择合适的地点圈地（资源丰富、面积大）

**游戏平衡**:
- 最小面积限制（100m²）: 防止微小领地刷资源
- 最大面积限制（1,000,000m²）: 防止垄断
- 防作弊机制: 检测GPS漂移、速度异常
- 重叠检测: 不允许侵占他人领地

**玩家策略**:
- **城市玩家**: 圈小面积高价值地块（POI密集区）
- **郊区玩家**: 圈大面积领地（开阔空间）
- **探险玩家**: 去未开发区域抢占先机

#### 4.1.2 POI发现与搜刮
**核心机制**: Walk to Earn - 走到真实地标就能获得资源

**发现机制**:
- 玩家进入POI 100米范围内触发"发现"
- 弹出提示框显示POI名称和类型
- 点击"搜刮"按钮获取资源

**资源映射逻辑**:
| POI类型 | 现实场景 | 游戏资源 | 设计理由 |
|---------|---------|---------|---------|
| 医院/药店 | 医疗机构 | 药品（急救包、抗生素） | 现实逻辑对应 |
| 超市/餐厅 | 食品供应点 | 食物（罐头、饮用水） | 生存必需品 |
| 工厂/工地 | 工业区 | 金属材料 | 建造材料 |
| 公园/林地 | 绿地 | 木材 | 自然资源 |
| 银行/珠宝店 | 高价值场所 | 稀有材料 | 稀缺资源，增加探索动力 |

**冷却时间设计**:
- 每个POI搜刮后进入6小时冷却
- 目的: 鼓励玩家探索更多地点，而非刷同一个点
- 策略: 规划路线，最大化探索效率

**奖励随机性**:
- 基础奖励: 固定数量（如罐头×10-30）
- 随机奖励: 小概率掉落稀有物品（5%概率）
- 暴击奖励: 极小概率双倍奖励（1%概率）

**玩家心理**:
- **收集乐趣**: 像宝可梦GO一样，每个地点都可能有惊喜
- **路线规划**: 玩家会规划高效的探索路线
- **意外惊喜**: 日常通勤也能顺便搜刮资源

### 4.2 资源管理系统

#### 4.2.1 背包系统
**设计目标**: 让资源管理成为策略选择，而非无脑囤积

**容量限制**:
- 初始容量: 100格
- 扩容方式:
  - 建造仓库: +50格/个（最多3个仓库）
  - 购买月卡: +50格
  - 成就奖励: +20格

**物品分类**:
1. **食物类**: 罐头、饮用水、食材
2. **材料类**: 木材、金属、石头、布料
3. **工具类**: 工具箱、设备零件
4. **稀有类**: 卫星模块、芯片、珍稀矿石

**堆叠规则**:
- 常见物品（食物、基础材料）: 最多堆叠99个
- 稀有物品（卫星模块）: 最多堆叠10个
- 特殊物品（建筑图纸）: 不可堆叠

**游戏平衡**:
- 容量限制迫使玩家做选择（带食物还是材料？）
- 鼓励玩家定期清理背包（使用、交易、丢弃）
- 仓库建筑成为重要的早期目标

#### 4.2.2 资源消耗循环
**设计理念**: 资源要有消耗出口，否则会通货膨胀

**消耗途径**:
1. **建造建筑**: 一次性消耗大量资源
2. **生存体征**: 持续消耗食物和水
3. **建筑升级**: 消耗稀有资源
4. **维修损坏**: 消耗材料
5. **交易成本**: 可选的交易税（未来版本）

**资源稀缺性设计**:
- **常见资源**（木材、食物）: 容易获得，大量消耗
- **稀有资源**（卫星模块）: 难以获得，用于高级建筑
- **通过稀缺性创造价值感和成就感**

### 4.3 建造系统

#### 4.3.1 建筑类型与功能
**设计理念**: 每个建筑都有明确的功能，不做无意义的装饰品

| 建筑名称 | 类型 | 等级 | 功能 | 建造成本 |
|---------|-----|------|------|---------|
| **篝火** | 生存 | Tier 1 | 提供温暖，解锁建造菜单 | 石头×20, 木材×30 |
| **庇护所** | 生存 | Tier 1 | 恢复生命值 | 木材×50, 布料×20 |
| **小仓库** | 储存 | Tier 1 | 背包容量+50 | 木材×80, 金属×30 |
| **农田** | 生产 | Tier 2 | 每24h产出食物×20 | 木材×100, 种子×10 |
| **工作台** | 生产 | Tier 2 | 合成高级物品 | 金属×150, 工具×20 |
| **太阳能板** | 能源 | Tier 2 | 为建筑提供能源 | 金属×200, 芯片×5 |
| **采矿场** | 生产 | Tier 3 | 每24h产出金属×30 | 金属×300, 卫星模块×2 |
| **卫星天线** | 通讯 | Tier 3 | 解锁全球通讯 | 卫星模块×10, 芯片×20 |

#### 4.3.2 建筑状态机
**设计目标**: 建筑不是摆设，需要玩家维护和管理

**状态流转**:
```
建造中 (施工期间)
    ↓
待激活 (需要能源激活)
    ↓
运行中 (正常工作) ⇄ 损坏 (需要维修)
    ↓
已拆除 (回收部分资源)
```

**状态详解**:
- **建造中**:
  - 持续时间: 根据建筑等级（Tier 1: 即时, Tier 2: 5分钟, Tier 3: 30分钟）
  - 可取消建造，退还50%资源

- **待激活**:
  - 需要消耗能源（食物、电力）
  - 激活后进入运行状态

- **运行中**:
  - 正常提供功能
  - 随机事件可能导致损坏（每24小时5%概率）
  - 可以升级（消耗资源提升效率）

- **损坏**:
  - 停止工作，需要维修
  - 维修成本: 建造成本的30%

**玩家互动**:
- 定期检查建筑状态
- 提前准备维修资源
- 升级建筑提升产出效率

#### 4.3.3 建筑布局策略
**空间限制**: 每个领地最多建造50个建筑

**布局策略**:
- **生存优先**: 先建篝火和庇护所（基础生存）
- **扩容仓库**: 早期建仓库扩容（提升探索效率）
- **生产建筑**: 中期建农田和采矿场（被动收入）
- **高级设施**: 后期建卫星天线和发电站（解锁高级功能）

### 4.4 交易系统

#### 4.4.1 以物易物机制
**设计理念**: 在新世界，货币失去意义，物物交换才是王道

**交易流程**:
1. 玩家发布挂单: "我用罐头×20换卫星模块×1"
2. 附近玩家（100米内）看到挂单
3. 接受交易，双方资源即时扣除和增加
4. 交易完成，双方获得成就进度

**位置限制设计**:
- 只能看到100米内且"允许交易"的领地挂单
- 目的: 鼓励玩家外出探索，增加偶遇感
- 策略: 在热门地点（商场、地铁站）发布挂单

**交易安全**:
- 原子性事务: 要么全成功，要么全失败
- 并发锁: 同一挂单只能被一个人接受
- 防刷机制: 每日交易次数上限（免费20次，月卡无限）

#### 4.4.2 交易经济系统
**物品价值参考**:
| 物品 | 稀有度 | 参考价值 |
|------|-------|---------|
| 罐头 | 常见 | 1 |
| 木材 | 常见 | 1 |
| 金属 | 常见 | 2 |
| 药品 | 稀有 | 5 |
| 芯片 | 稀有 | 10 |
| 卫星模块 | 史诗 | 50 |

**玩家驱动经济**:
- 供需关系决定价格（新区木材便宜，老区金属便宜）
- 稀有物品产生交易需求
- 玩家成为"商人"角色（囤货、倒卖）

### 4.5 社交系统

#### 4.5.1 通讯设备等级
**设计目标**: 通过设备等级限制通讯范围，创造付费点和成长目标

| 设备 | 接收范围 | 发送能力 | 获取方式 |
|------|---------|---------|---------|
| 收音机 | 无限 | 不可发送 | 默认拥有 |
| 对讲机 | 3km | 可发送 | 任务解锁/付费$2.99 |
| 营地电台 | 10km | 可发送 | 付费$9.99 |
| 卫星通讯 | 全球 | 可发送 | 月卡或付费$19.99 |

**游戏进程设计**:
- 新手: 只能听广播（接收信息）
- 进阶: 解锁对讲机（开始社交）
- 资深: 购买营地电台（扩大影响力）
- 领主: 解锁卫星通讯（全球联络）

#### 4.5.2 基于位置的社交
**公共频道**:
- 3km范围内的玩家共享聊天室
- 实时消息推送
- 用例: "附近有物资点吗？" "求交易卫星模块"

**自定义频道（群组）**:
- 玩家创建私人频道
- 邀请好友加入
- 不受距离限制（但需要对讲机以上设备）
- 用例: 组建联盟、交易群、区域联盟

**PTT呼叫（对讲机模式）**:
- 长按发送语音消息
- 模拟真实对讲机体验
- 增加游戏沉浸感

### 4.6 生存系统

#### 4.6.1 生存体征设计
**核心机制**: 时间衰减 + 资源消耗 = 持续游玩动力

**三大体征**:
1. **核心生命值** (0-100)
   - 降至0则角色死亡
   - 死亡惩罚: 丢失背包20%物品，24小时复活冷却

2. **饱食度** (0-100%)
   - 衰减速度: -5%/小时
   - < 50%时，生命值开始下降（-2/小时）
   - > 80%时，获得增益：探索效率+20%

3. **水分** (0-100%)
   - 衰减速度: -10%/小时（比饱食度快）
   - < 50%时，生命值开始下降（-3/小时）
   - > 80%时，获得增益：建造速度+20%

**恢复方式**:
| 物品 | 恢复效果 | 获取难度 |
|------|---------|---------|
| 罐头食品 | +20饱食度 | 容易（超市POI） |
| 饮用水 | +30水分 | 容易（超市POI） |
| 急救包 | +50生命值 | 中等（医院POI） |
| 营养餐 | +50饱食度+30水分 | 困难（需合成） |

**游戏平衡**:
- **轻度玩家**: 每天登录10分钟，吃1次食物即可
- **中度玩家**: 每天探索1小时，需要吃3-5次
- **重度玩家**: 长时间在线，需要大量食物和水

**设计意图**:
- 创造资源消耗循环（不会通货膨胀）
- 鼓励持续游玩（不能"放置"几天不管）
- 增加紧迫感和生存感
- 付费点: 卖"减缓衰减"道具或"营养餐包"

#### 4.6.2 死亡与复活机制
**死亡触发**:
- 生命值降至0

**死亡惩罚**:
- 丢失背包内20%物品（随机）
- 角色进入"濒死"状态
- 需要等待24小时或使用"复活币"（付费道具）

**复活方式**:
1. 等待24小时自然复活（免费）
2. 使用复活币立即复活（$0.99）
3. 好友救援（需要好友在你领地100米内）

**设计理由**:
- 死亡要有惩罚，增加生存紧张感
- 但不能太重，避免玩家弃坑
- 提供付费快速复活选项（变现点）

### 4.7 成长系统

#### 4.7.1 成就系统
**设计目标**: 通过成就引导玩家体验所有玩法，同时提供炫耀资本

**成就分类**:

**新手成就**（引导玩法）:
- 末世拾荒者: 完成第一次搜刮 → 金币×100
- 新手领主: 完成第一次圈地 → 金币×200
- 初次建造: 建造第一个建筑 → 金币×150
- 社交开端: 发送第一条消息 → 金币×50
- 首次交易: 完成第一次交易 → 金币×200

**进阶成就**（挑战目标）:
- 探索者: 搜刮50个POI → 金币×500, 称号"探索者"
- 建筑大师: 建造10个建筑 → 金币×1000
- 交易达人: 完成100次交易 → 金币×1500
- 领地扩张: 拥有5块领地 → 金币×2000

**稀有成就**（炫耀资本）:
- 传奇拾荒者: 搜刮1000个POI → 金币×10000, 传说称号
- 废土领主: 领地总面积超过100万m² → 金币×20000, 专属头像框
- 交易大亨: 交易总额超过10万单位 → 金币×15000
- 幸存者: 存活365天 → 金币×50000, 专属纪念徽章

**成就奖励**:
- 金币（用于购买资源包）
- 称号（显示在个人主页）
- 头像框/徽章（稀有成就专属）
- 解锁特殊建筑图纸

**EventBus事件驱动**:
- 玩家完成行为 → 发出事件 → 成就系统监听 → 自动检测达成
- 解耦合设计，易于扩展新成就

#### 4.7.2 排行榜系统
**设计目标**: 激发竞争欲，促进分享传播

**榜单类型**:
1. **领地面积榜** - 谁的地盘最大
2. **探索废墟榜** - 谁搜刮的POI最多
3. **建筑数量榜** - 谁建造的建筑最多

**更新机制**:
- 每5分钟更新一次（数据库缓存）
- 显示Top 100玩家
- 显示自己的排名（即使不在Top 100）

**社交功能**:
- 点击玩家头像查看资料
- 分享自己的排名到社交媒体
- 挑战榜单第一名（发起友好竞赛）

**赛季机制**（未来版本）:
- 每月重置榜单
- 赛季奖励（前10名获得专属称号和资源）

## 5. 游戏经济系统

### 5.1 资源产出与消耗平衡

**每日资源产出**（假设中度玩家）:
- 探索5个POI: 食物×50, 材料×30, 稀有×2
- 生产建筑产出: 食物×20, 金属×10

**每日资源消耗**:
- 生存体征: 食物×30, 水×40
- 建造1个建筑: 材料×80-200
- 交易支出: 视情况

**经济设计原则**:
- 保持轻微稀缺（产出略小于需求）
- 鼓励玩家持续探索和交易
- 稀有资源极度稀缺，创造长期目标

### 5.2 货币系统

**金币**:
- 获取: 完成成就、每日任务
- 用途: 购买资源包（游戏内商店）
- 不可交易（避免RMT黑市）

**资源包定价**（金币商店）:
- 小型物资包: 500金币 → 食物×50, 木材×30
- 中型物资包: 1500金币 → 稀有材料×10
- 大型物资包: 5000金币 → 卫星模块×2

### 5.3 付费设计

**订阅制（月卡）**:
- **开拓者月卡** ($9.99/月):
  - 无限探索次数（免费玩家每日20次）
  - 探索范围扩大至5km
  - 背包容量+50
  - 每日登录奖励×2

- **领主通行证** ($19.99/月):
  - 月卡所有特权
  - 解锁所有通讯设备
  - 专属建筑皮肤（希望主题）
  - 交易手续费减免

**一次性购买**:
- 初级物资包 ($0.99): 食物×50, 木材×30
- 中级物资包 ($2.99): 稀有材料×10, 芯片×5
- 高级物资包 ($9.99): 卫星模块×3, 随机史诗建筑图纸
- 复活币×5 ($2.99): 立即复活，无惩罚
- 对讲机解锁 ($2.99): 永久解锁对讲机

**付费设计原则**:
- 非Pay-to-Win（付费不能直接获得绝对优势）
- 提供便利而非必需（免费玩家也能完整体验）
- 订阅制提供持续价值
- 小额付费降低决策门槛

## 6. 游戏进程设计

### 6.1 新手引导（前30分钟）

**第1步**: 登录注册
- 选择登录方式
- 自动进入新手教程

**第2步**: 首次圈地
- 引导玩家在起始位置附近圈一块小地（教学领地）
- 自动命名为"避难所"
- 奖励: 金币×200, 基础资源×50

**第3步**: 首次探索
- 引导玩家找到最近的POI
- 完成首次搜刮
- 获得食物和材料
- 解锁成就"末世拾荒者"

**第4步**: 首次建造
- 引导建造"篝火"（资源已足够）
- 解释建筑功能
- 解锁成就"初次建造"

**第5步**: 生存体征
- 展示饱食度和水分
- 引导使用罐头恢复
- 解释生存机制

**教学设计原则**:
- 分步骤，每次只教一个功能
- 即时奖励，强化正反馈
- 可跳过（老玩家小号）

### 6.2 早期游戏（第1-7天）

**第1天目标**:
- 圈下第一块真正的领地（选址重要）
- 搜刮5个POI，熟悉资源类型
- 建造篝火和庇护所
- 加入公共频道，感受社交

**第2-3天目标**:
- 建造小仓库（扩容背包）
- 探索更远的地方（发现新POI）
- 尝试发布交易挂单
- 完成"探索者"成就（搜刮10个POI）

**第4-7天目标**:
- 解锁对讲机（任务或付费）
- 建造生产建筑（农田）
- 圈占第二块领地
- 进入排行榜（哪怕只是第100名）

**留存设计**:
- 每日登录奖励
- 每日任务（搜刮3个POI，建造1个建筑）
- 社交互动（聊天、交易）
- 成就进度可视化

### 6.3 中期游戏（第8-30天）

**核心玩法**:
- 扩张领地（3-5块领地）
- 建造高级建筑（Tier 2）
- 活跃交易（成为区域商人）
- 攀升排行榜（进入Top 50）

**社交深化**:
- 加入玩家频道（联盟）
- 建立交易网络
- 参与区域事件（未来版本）

### 6.4 后期游戏（30天+）

**长期目标**:
- 完成稀有成就
- 建造所有Tier 3建筑
- 排行榜Top 10
- 成为区域领主（影响力最大的玩家）

**内容更新**:
- 新建筑（每月更新）
- 限时活动（节日主题）
- 赛季排行榜（月度重置）

## 8. 技术设计

### 8.1 防作弊机制

**GPS作弊检测**:
- 速度检测: 速度 > 50km/h标记异常
- 轨迹合理性: 检测瞬移、跳跃
- 设备检测: 检测模拟器、越狱设备
- 行为分析: 异常探索频率

**交易作弊防范**:
- IP限制: 同一IP多账号交易限制
- 频率限制: 每日交易上限
- 价值检测: 明显不等价交易警告

**封禁机制**:
- 轻度作弊: 警告 + 7天禁止圈地
- 中度作弊: 14天封禁
- 重度作弊: 永久封禁

### 8.2 性能优化

**地图渲染**:
- 只渲染视野内的领地和建筑
- LOD（细节层次）: 远距离简化模型
- 缓存机制: 缓存已加载的数据

**数据同步**:
- 本地优先: 先更新本地，后台同步云端
- 增量同步: 只同步变化的数据
- 压缩传输: 减少流量消耗

### 8.3 服务器架构

**数据库设计**:
- PostgreSQL + PostGIS（地理数据）
- Redis缓存（排行榜、热点数据）
- Supabase Realtime（实时消息）

**扩展性**:
- 分区分服（每个城市一个服务器集群）
- 负载均衡
- CDN加速（静态资源）

## 11. 核心设计哲学

### 11.1 设计原则
1. **真实世界互动**: 游戏与现实深度结合
2. **公平游戏**: 付费提供便利，但不能Pay-to-Win
3. **社交驱动**: 鼓励玩家互动合作
4. **长期养成**: 提供持续数月的成长目标
5. **希望主题**: 传递积极向上的价值观

### 11.2 玩家体验目标
- **新手**: 10分钟上手，30分钟理解核心玩法
- **中度玩家**: 每日30-60分钟游玩，持续2-3个月
- **重度玩家**: 深度策略，持续6个月以上

### 11.3 成功指标
- 次日留存 > 40%
- 7日留存 > 25%
- 付费转化率 > 5%
- 玩家日均游戏时长 > 30分钟
